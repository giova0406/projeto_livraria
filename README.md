# 📚 Projeto Livraria

Projeto de web scraping em Python que coleta títulos e preços de livros do site http://books.toscrape.com/ e exporta os dados para um arquivo CSV.

## 🚀 Tecnologias utilizadas
- Python
- Requests
- BeautifulSoup
- CSV

## ▶️ Como executar o projeto
1. Clonar o repositório ou baixar os arquivos
2. Criar e ativar um ambiente virtual
3. Instalar as dependências:
   pip install -r requirements.txt
4. Executar o arquivo:
   python main.py

## 📊 Resultado
O projeto gera um arquivo chamado `relatorio_livros.csv` contendo os dados extraídos do site.
